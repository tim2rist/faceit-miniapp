# Руководство по развертыванию (Deployment Guide) — Faceit CS2 Telegram Bot

Это пошаговое руководство предназначено для системного администратора компьютерного клуба (игрового зала). В нем описано, как установить, запустить и настроить автозапуск Telegram-бота для отслеживания статистики Faceit CS2 на локальном сервере или администраторском ПК под управлением Windows / Linux.

---

## 1. Системные требования и предварительная подготовка (Prerequisites)

Для работы бота требуется среда выполнения **Node.js**.

1. Перейдите на официальный сайт [Node.js](https://nodejs.org/).
2. Скачайте и установите **LTS-версию** (рекомендуемую для большинства пользователей).
3. Во время установки оставьте все настройки по умолчанию (убедитесь, что установлена галочка добавления Node в `PATH` для системных переменных).
4. После завершения установки откройте консоль (Командную строку Windows или PowerShell) и проверьте корректность установки:
   ```bash
   node -v
   npm -v
   ```
   *Обе команды должны вывести установленные версии (например, `v20.x.x` и `10.x.x`).*

## 2. Перенос файлов проекта (Transferring Files)

Некоторые конфигурационные файлы и локальные базы данных содержат конфиденциальные ключи доступа, поэтому они игнорируются системой контроля версий Git. Вам необходимо перенести их вручную:

1. **Основные файлы**: Склонируйте репозиторий с GitHub или скопируйте файлы проекта в рабочую директорию на сервере (например, `C:\faceit-bot` или `/opt/faceit-bot`).
2. **Конфиденциальные файлы (КРИТИЧЕСКИ ВАЖНО)**:
   * **`.env`**: Скопируйте файл конфигурации `.env` с исходного компьютера. В нем хранятся секретный токен Telegram-бота (`TELEGRAM_TOKEN`) и API-ключ Faceit (`FACEIT_API_KEY`). **Никому не передавайте этот файл!**
   * **`database.sqlite`**: Скопируйте файл базы данных SQLite из корня проекта. Он содержит список всех привязанных игроков и чатов. Если вы не скопируете этот файл, бот запустится с чистой базой данных, и пользователям придется привязывать свои аккаунты заново через команду `/link`.

Убедитесь, что файлы `.env` и `database.sqlite` лежат в корневой папке проекта рядом с файлом `bot.js`.

## 3. Установка зависимостей (Installation)

Откройте терминал/консоль, перейдите в папку с проектом и выполните установку библиотек:

```bash
npm install
```
*Команда автоматически загрузит все необходимые пакеты (`telegraf`, `axios`, `dotenv`, `node-cron`), указанные в `package.json`.*

## 4. Автозапуск и работа в фоновом режиме (PM2)

Чтобы бот работал 24/7, не закрывался при закрытии окна консоли и автоматически запускался при перезагрузке компьютера, мы используем менеджер процессов **PM2**.

### Шаг 1. Глобальная установка PM2
Выполните установку PM2 глобально в системе:
```bash
npm install -g pm2
```

### Шаг 2. Запуск бота под управлением PM2
Запустите бота под управлением PM2, присвоив ему имя `faceit-bot`:
```bash
pm2 start bot.js --name faceit-bot
```

### Шаг 3. Настройка автозапуска при перезагрузке системы

#### Инструкция для Windows (основная ОС администраторов):
1. Установите глобальный модуль для настройки службы автозапуска PM2 в Windows:
   ```cmd
   npm install -g pm2-windows-startup
   pm2-startup install
   ```
2. Сохраните текущий список активных процессов PM2, чтобы служба знала, какие скрипты автоматически поднимать при старте ОС:
   ```cmd
   pm2 save
   ```

#### Инструкция для Linux (если сервер на Linux):
1. Сгенерируйте скрипт автозапуска:
   ```bash
   pm2 startup
   ```
   *(Терминал выведет специальную команду, которую нужно скопировать и запустить с правами `sudo`).*
2. Сохраните список процессов:
   ```bash
   pm2 save
   ```

## 5. Обслуживание и мониторинг (Maintenance)

Основные команды PM2 для администрирования бота:

* **Просмотр логов (вывод консоли в реальном времени)**:
  ```bash
  pm2 logs faceit-bot
  ```
  *(Используйте для отслеживания ошибок инициализации или логов cron-планировщика).*
* **Просмотр статуса активных процессов**:
  ```bash
  pm2 status
  ```
* **Перезапуск бота** (например, после обновления файлов или `.env`):
  ```bash
  pm2 restart faceit-bot
  ```
* **Остановка бота**:
  ```bash
  pm2 stop faceit-bot
  ```
* **Удаление процесса из панели PM2**:
  ```bash
  pm2 delete faceit-bot
  ```
* **Интерактивный графический мониторинг (нагрузка на CPU/RAM)**:
  ```bash
  pm2 monit
  ```
